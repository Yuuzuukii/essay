# 参考文献（References）

出典：`template/guide.tex`「執筆上の注意 > 文献」「スタイルの使い方 > 参考文献」。

## 位置

- 文献表は **本文に続いて**置く（付録がある場合は本文と付録の間）。

## 本文中の引用（ガイドの方式）

- 本文との対応は `[1]`, `[2]` のように **[ ]付きの通し番号**。
- 挿入位置は、該当文章の切れ目、または人名その他の単語に続けて入れる。

## 文献表に書く項目

- 著者名
- 表題
- 雑誌名（または書名）
- 巻
- 年号
- ページ 等

（検索しやすくするため）雑誌の「号」や「月」も入れると良い場合がある。

## BibTeX推奨（`kuisthesis`想定）

- 文献DB（`.bib`）を作り、`kuisunsrt` または `kuissort` を使う
  - `kuisunsrt`：出現順
  - `kuissort`：著者名アルファベット順
- BibTeXが使えない場合は `thebibliography` 環境で同形式に合わせる

```tex
\bibliographystyle{kuisunsrt} % or kuissort
\bibliography{<your-bib>}
```

## チェック

- 重要文献に絞れている（無意味な羅列になっていない）
- 引用しているのに文献表に無い／文献表にあるのに本文で未引用がない
- 出典の体裁が統一されている（著者・年・ページ等の欠落がない）
