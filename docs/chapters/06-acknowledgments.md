# 謝辞（Acknowledgments）

出典：`template/guide.tex`「論文の構成 > 結論」「スタイルの使い方 > 謝辞」。

## 位置づけ

- 結論の後に置くのが望ましい（慣習）。
- 指導・助言・援助を受けた人々への礼儀として記す。

## 書くこと

- 指導教員・共同研究者・支援者（研究室・資金・設備等）への感謝
- 具体的な支援内容は書きすぎない（簡潔でよい）

## LaTeX（`kuisthesis`想定）

- `\acknowledgments` に続けて本文を書く（見出しと目次登録は自動）

```tex
\acknowledgments
（謝辞本文）
```

## チェック

- 名前・所属・役職の誤記がない
- 砕けすぎた表現になっていない（論文全体の文体と整合）
