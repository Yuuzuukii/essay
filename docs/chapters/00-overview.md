# 全体構成（本文・付録の考え方）

出典：`template/guide.tex`「論文の構成」「執筆上の注意」。

## 必須の構成要素（基本）

報告書／論文は基本的に以下で構成する。

1. 内容梗概（和文・英文）
2. 目次
3. 本文（緒論＝序論／本論／結論）
4. （必要に応じて）付録
5. （本文末）参考文献
6. （結論の後）謝辞（慣習として推奨）

ポイント：

- **本文だけで完結**したものにする（付録の有無に関わらず）。
- 付録は本文の補足であり、**本文の理解に必須な中核**を押し出さない（押し出すなら本文へ）。
- 引用と創意（あなたの新規性）の境界を明確にし、**良心的に**書く。

## まず決めるべきこと（書き始める前）

- 想定読者（同分野の学部生？研究室内？審査員？）と前提知識
- 研究の「位置づけ・目的・性格」（緒論で言い切れる形）
- 主張（結論で言い切れる形）と、その根拠（本論の流れ）
- 図表の方針（図表比率の上限を意識し、代表のみを本文へ）
- 記号の方針（定義の場所／記号表を入れるか）

## 章立ての指針（本論を壊さないためのチェック）

- 章・節の粒度を揃える：短すぎ／長すぎを避け、**適切な長さに分割**する。
- 見出しは内容を反映：章題が「何が書いてあるか」を表すようにする。
- 章間の接続：各章が独立して読めつつ、**自然につながる**ように書く。
- 重要部分の強調：だらだら書かず、精疎の配分で**独創・要点**を前に出す。

## 体裁（執筆上の注意の要点）

- 図表は原則 **本文中に挿入**し、参照箇所の近くに置く（代表のみ本文、残りは付録でも可）。
- 表キャプションは上、図キャプションは下。図表だけ見ても概ね意味が分かるようにする。
- 記号は明確に定義し、必要なら「記号表」を挿入する。
- 引用文献は本文末（付録がある場合は本文と付録の間）にまとめる。
