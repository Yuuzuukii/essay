# 付録（Appendix）

出典：`template/guide.tex`「論文の構成 > 付録」「スタイルの使い方 > 付録」。

## 目的（本文の補足）

- 本文の内容を補足し、より充実させるために付ける。
- ただし、**本文だけで完結**している必要がある（付録がないと論旨が成立しないのはNG）。

## 付録に回す典型（ガイドの例）

- 複雑で長い数式の誘導（本文の可読性を下げる詳細）
  - 本文と付録の対応（参照関係）を明示する
- 必要な定理の証明（ただし証明自体が研究目的なら本文へ）
- 大量の観測データや数値計算結果（本文は代表、全体は付録）
- 長いプログラムのソースリスト
- 大量データ・長大プログラムは別冊付録も可（体裁は本文に準じるのが望ましい）
- 付録だけで概ね意味が分かるよう、標題と簡単な説明を付ける

## LaTeX（`kuisthesis`想定）

- `\appendix` または `\Appendix` の後に書く
  - `\appendix`：付録ページ番号を印字しない
  - `\Appendix`：A-1, A-2…のページ番号を印字
- どちらもオプションで付録全体の見出しを付けられる

例：

```tex
\Appendix[付録：追加実験結果]
\section{...}
```

## チェック

- 本文から付録への参照があり、付録も“読まれる前提”で整理されている
- 付録に逃げて良いもの／本文に残すべきものの境界が妥当
