# 内容梗概（Abstract / 概要）

出典：`template/guide.tex`「論文の構成 > 内容梗概」「執筆上の注意 > 用語」「スタイルの使い方 > 内容梗概」。

## 目的（何のために書くか）

- 論文が **何を目的**とし、**どの結論**に達したかを短い文章で示す。
- 本文の単なる圧縮ではなく、研究の中心部分を抽出して書く。

## 分量・言語（最低限守ること）

- （本文が和文でも英文でも）**和文・英文の内容梗概を両方**付ける。
- 両方とも **2ページ以内**（ガイドの規定）。

## 書くべき要素（構造テンプレ）

1. 背景（1–2文）：何が問題か、なぜ重要か
2. 目的：何を明らかに／実現しようとしたか
3. 提案・手法：何をどうしたか（核だけ）
4. 評価・結果：何がどれだけ良くなったか／何が分かったか
5. 結論：結局何が言えるか、限界や適用範囲（必要なら）

## よくあるNG（ガイドで注意されている点）

- 内容梗概の大半が緒論（序論）の話になっている（目的から外れる）。
- 「本文の導入」になっていて、結論や成果が薄い。

## LaTeX（`kuisthesis`想定）の書き方

- 和文：`jabstract` 環境
- 英文：`eabstract` 環境
- 記述順は、本文言語に合わせて「本文の言語→もう片方」の順が自然

最小形（概形）：

```tex
\begin{jabstract}
（和文概要）
\end{jabstract}
\begin{eabstract}
(English abstract)
\end{eabstract}
```

## 最終チェック

- 目的・手法・結果・結論が1回ずつ登場している
- 数値／比較／観察（何らかの根拠）が入っている
- 2ページ以内（和・英それぞれ）
