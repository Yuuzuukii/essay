# thesis/guide まとめ（docs）

この `docs/` は `template/guide.tex`（「特別研究報告書・修士論文 執筆の手引」）を読み取り、**各章（構成要素）ごとの「何を書くべきか／何を避けるべきか」**を、執筆時に参照しやすい形で整理したメモです。  
※最終的な提出規定は所属専攻・研究室の指示を優先してください。

## 章別ガイド

- `docs/chapters/00-overview.md`：全体構成の方針（本文＝緒論・本論・結論、付録の位置づけ）
- `docs/chapters/01-abstract.md`：内容梗概（和文・英文）
- `docs/chapters/02-toc.md`：目次
- `docs/chapters/03-introduction.md`：緒論（序論）
- `docs/chapters/04-mainbody.md`：本論
- `docs/chapters/05-conclusion.md`：結論
- `docs/chapters/06-acknowledgments.md`：謝辞
- `docs/chapters/07-references.md`：参考文献
- `docs/chapters/08-appendix.md`：付録

## レビュー用テンプレ

- `docs/review_temp.md`：レビュー記入用（章別コメント／体裁チェックの欄つき）
