# 卒論・修論 書式レビュー用チェックリスト（Copilot用）

## レビュー指示（Copilotへの依頼文）
以下のチェックリストに基づいて，本ドキュメント（卒論／修論）の**書式・体裁のみ**を厳密にレビューせよ．  
**内容の新規提案や研究的改善は不要**．  
違反・不備がある箇所については，

- 問題点
- 該当箇所
- 修正指示（具体的）

を簡潔に列挙せよ．  
「問題なし」の場合もその旨を明示せよ．

---

## 1. 標題（タイトル）

- [ ] タイトルが以下の形式に沿っているか  
  - 「〜（手法）〜による〜（主題）〜」  
  - 「〜（手法）〜に基づく〜（主題）〜」  
  - 「〜（手法）〜を用いた〜（主題）〜」  
  - 「〜（応用先）〜のための〜（主題）〜」
- [ ] **主題に対して何をしたか／何に応用したか**が明確か
- [ ] 英語タイトルの場合：
  - [ ] 前置詞・冠詞以外はすべて **大文字始まり（Capitalize）** か

---

## 2. 内容梗概（Abstract / 概要）

- [ ] 日本語・英語それぞれ **2ページ以内**に収まっているか
- [ ] 貢献部分に以下のいずれかが明記されているか  
  - [ ] 定量評価（数値・比較・指標）
  - [ ] 定性評価（分析結果・考察）
- [ ] フォント・文字サイズが **テンプレート準拠** か
- [ ] 英語タイトルの Capitalize ルールが守られているか

---

## 3. 文章表現

- [ ] 文体が **「である調」** で統一されているか
- [ ] 冗長表現がないか  
  - NG例：「〜できるということである」
  - OK例：「〜できる」
- [ ] 「つまり，」が単なる繰り返しになっていないか
- [ ] 句読点が以下で統一されているか  
  - **全角「，」「．」**
- [ ] 英語論文の場合：
  - [ ] 第一人称が **I ではなく We**
  - [ ] 不必要な受動態を避け，**無生物主語**が使われているか

---

## 4. 箇条書き

- [ ] 箇条書きが **過剰に使用**されていないか
- [ ] パラメータ・条件列挙は **表に整理**されているか

---

## 5. 図表

- [ ] 表キャプションが **表の上** にあるか
- [ ] 図キャプションが **図の下** にあるか
- [ ] 図表が **ページ上部または下部** に配置されているか
- [ ] 図表が **ページをまたいでいない** か
- [ ] 図表・キャプションが **中央揃え** か
- [ ] 図表内フォントサイズが **本文よりやや小さい** か
- [ ] 図に **不要な枠線** がないか
- [ ] 表の列名が **中央揃え＋太字** になっているか

※ Word 使用時の注意  
- [ ] 図表の「文字列の折り返し」が「行内」ではなく **「四角」** か  
- [ ] レイアウト崩れが発生していないか  
（※ 可能なら LaTeX 使用を推奨）

---

## 6. 数式

- [ ] Word 使用時，**数式エディタ**を使用しているか
- [ ] 変数が **小文字イタリック** か（例：x, y）
- [ ] 添字が正しく下付・上付になっているか（例：x₁, xᵢⱼ）
- [ ] 集合が **大文字イタリック** か（例：X, Y）
- [ ] ベクトルが **太字小文字イタリック** か（例：𝐱）
- [ ] 定数が **大文字** か（例：TIME）
- [ ] 述語が **キャメル形式・イタリック** か  
  - 例：isSupervisedBy(x, MURAKAMI)

---

## 7. 引用・参考文献

- [ ] 引用が原則 **文末** に入っているか
- [ ] 文末引用が **句点の前** にあるか  
  - 正：「〜〜［1］．」
- [ ] Word 使用時，「相互参照」による自動採番を使っているか
- [ ] 参考文献数が要件を満たしているか  
  - 卒論：**5件以上（目標7件）**  
  - 修論：**10件以上（目標15件）**
- [ ] 教科書・入門書ではなく **論文・専門書** が中心か
- [ ] Webページ引用時：
  - [ ] URLが **脚注** に記載されているか
- [ ] 参考文献フォーマットが以下に完全一致しているか
